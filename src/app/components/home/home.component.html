H2 Database

H2 Database is an In-Memory Database that allows you to achive CRUD operations during development. It is a Full-featured Relational Database Management System (RDBMS) with a small footprint, either embedded into Java application or used as a database server. H2 database is very simple to setup. It simply needs its dependency added to you rproject then you can access it when you application is running. This database Engine is primarily used for running test cases by the development teams
